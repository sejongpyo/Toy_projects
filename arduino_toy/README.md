# arduino_toy
- 팀원: Sejong Pyo (수업 개인 프로젝트)
- 참여 기간: 2021.03.02 ~ 2020.06.21
- 목표: 아두이노를 활용한 게임 제작
- 주요 작업 내용: 장애물 피하기 게임, 진동 인식을 이용한 캐릭터 조작

### Work Flow
1. 장애물 피하기 게임 (running man game)
2. piezo vibration sensor
- 노크 방식으로 게임 내 케릭터 조작
3. door lock opening
- servo motor를 사용해 일정 점수 득점 시 문 열기
4. piezo buzzer를 사용해 효과음 송출
